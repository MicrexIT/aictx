When resolving Dependabot pnpm update conflicts, lockfile-level overrides in pnpm-lock.yaml must be mirrored in package.json under pnpm.overrides. Otherwise this repo's CI command `pnpm install --frozen-lockfile` fails with ERR_PNPM_LOCKFILE_CONFIG_MISMATCH. Preserve Dependabot's transitive/security updates by adding or keeping the matching package-level overrides, then validate with `pnpm install --lockfile-only` and `pnpm install --frozen-lockfile`.
