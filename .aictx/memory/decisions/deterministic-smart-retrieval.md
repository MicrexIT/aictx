# Deterministic smart retrieval

Aictx load and search support optional retrieval hints for files, changed files, symbols, subsystems, and Git history windows. The generated index derives file, commit, and facet link tables plus recent Git file-change metadata from canonical memory and relation evidence. Context packs can render Architecture Snapshot, Rationale Gaps, and Linked History sections; Git history is treated as evidence that files changed, not as invented rationale for why they changed.
