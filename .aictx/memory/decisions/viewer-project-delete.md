The local viewer supports explicit project deletion through DELETE /api/projects/:registryId. Deletion derives the target .aictx/ path from project_root, permanently removes that .aictx/ directory without backup, unregisters the matching $AICTX_HOME/projects.json entry, treats already-missing .aictx/ directories as stale registry cleanup, and must not delete source files. The Svelte viewer requires a project-name confirmation before calling the destructive route.
