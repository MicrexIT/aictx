The local viewer exposes Delete memory on project cards only outside demo mode. The action requires typing the project id, calls DELETE /api/projects/:registryId with the viewer token, removes the project .aictx memory directory when present, unregisters the project from the viewer registry, refreshes the project list, and keeps ordinary repository/source files untouched. Public demo mode hides the UI action and the demo worker returns 403 for DELETE /api/projects/:id.
