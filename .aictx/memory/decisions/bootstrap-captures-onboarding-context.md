# Bootstrap captures onboarding context

First-run bootstrap suggestions now seed quick-win context from deterministic repo evidence: post-task verification workflows from relevant package scripts and explicit AGENTS/CLAUDE commands, code convention constraints from non-generated agent guidance, and product-feature concepts from README/docs feature bullets, package bin entries, CLI command descriptions, and page route files. Generated Aictx guidance blocks are ignored when mining AGENTS.md or CLAUDE.md so Aictx does not save its own boilerplate as project convention memory.
