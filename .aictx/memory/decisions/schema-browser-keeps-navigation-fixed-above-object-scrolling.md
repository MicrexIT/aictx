The local viewer schema browser uses a responsive browse/detail workspace. Compact object filters and trust status remain at the top. On desktop, the mini relation graph is a left context panel beside a wider object workspace; object rows remain stable and the selected object renders once as a right-side detail panel. The mini relation list shows only the first few direct relations plus an overflow count, avoiding internal graph scrollbars. On mobile, the relation overview disclosure starts open by default but renders as a compact graph plus View full graph action so the object list still has usable scroll space. Tapping an object enters a focused detail mode with a Back button, hiding filters, graph context, and the list while allowing natural page scrolling for long memory content.
