Aictx's near-term roadmap is now local MCP excellence followed by a host-neutral data-access layer. Local MCP exposes exactly load_memory, search_memory, inspect_memory, save_memory_patch, and diff_memory; save_memory_patch remains the only MCP write primitive. The new T001-T012 sequence first rebaselines specs/docs/tests and adds inspect_memory, then hardens local MCP and moves load/search/inspect/diff/save semantics behind shared data access. Remote MCP, OAuth, hosted sync, tenancy, billing, ChatGPT App SDK UI, embeddings, and cloud architecture remain deferred.
