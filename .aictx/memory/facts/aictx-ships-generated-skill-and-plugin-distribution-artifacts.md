Aictx keeps integrations/templates/agent-guidance.md as the source of truth and generates aictx-memory artifacts for Codex standalone skills, Codex plugins, and Claude Code plugins under integrations/codex/skills/aictx-memory, integrations/codex/plugins/aictx-memory, and integrations/claude/plugins/aictx-memory. These artifacts stay CLI-first and intentionally do not bundle MCP server configuration. The repo now also ships self-hosted marketplace catalogs: .agents/plugins/marketplace.json for Codex and .claude-plugin/marketplace.json for Claude Code. The public install docs intentionally stay short: Codex uses codex plugin marketplace add MicrexIT/aictx, then installs Aictx Memory from Codex Plugins; Claude Code uses /plugin marketplace add MicrexIT/aictx and /plugin install aictx-memory@aictx. Full publishing and official listing guidance lives in docs/src/content/docs/plugin-publishing.md.
