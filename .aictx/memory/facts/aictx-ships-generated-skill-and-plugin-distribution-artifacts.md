Aictx keeps integrations/templates/agent-guidance.md as the source of truth and generates PR-ready aictx-memory artifacts for Codex standalone skills, Codex plugins, and Claude Code plugins under integrations/codex/skills/aictx-memory, integrations/codex/plugins/aictx-memory, and integrations/claude/plugins/aictx-memory. These artifacts stay CLI-first and intentionally do not bundle MCP server configuration. Marketplace install commands target marketplace catalogs, not raw plugin folders: Codex uses codex plugin marketplace add/upgrade/remove, while Claude Code uses /plugin marketplace add/list/update/remove and claude plugin marketplace add/list/update/remove, then installs the plugin as aictx-memory@marketplace-name.
