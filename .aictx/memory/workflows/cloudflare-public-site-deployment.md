Deploy the public landing/blog site to Cloudflare Workers from the repository root using Workers static assets. The root `wrangler.jsonc` names the Worker `aictx`, sets `compatibility_date` to `2026-05-05`, and serves static assets from `./site/dist` with `html_handling: auto-trailing-slash` and `not_found_handling: 404-page`. Cloudflare Workers Builds should use build command `pnpm build:site` and deploy command `npx wrangler deploy` (or the package script `pnpm deploy:site:cloudflare`). The repo pins `wrangler` as a dev dependency and has `.node-version` set to Node 22. `pnpm build:site` and `pnpm exec wrangler deploy --dry-run` were verified on 2026-05-05. Do not deploy this project under the unrelated Vercel `rivista-rinascita` scope; the Vercel project was deleted and is superseded by Cloudflare.
