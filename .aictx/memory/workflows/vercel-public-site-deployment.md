Deploy the public landing/blog site to Vercel from the repository root. The root `vercel.json` sets `installCommand` to `npm install -g pnpm@10.11.0 && pnpm install --frozen-lockfile`, `buildCommand` to `pnpm build:site`, and `outputDirectory` to `site/dist`. `pnpm build:site` runs `astro build --root site`, producing the static landing page and blog output. The Vercel project is linked locally as `rivista-rinascita/aictx` but `.vercel/` is ignored. A production Vercel deployment was verified at `https://aictx-five.vercel.app/` and `/blog/` on 2026-05-05. Use Vercel custom domains for `aictx.dev`: apex/root can use an A record to Vercel's IP, and `www` can use a CNAME to Vercel's DNS target, following the exact records shown by Vercel.
